1- Na tag "input" com atributo type="text", para que serve o atributo "placeholder"?

R: O atributo placeholder é uma pequena frase ou palavra, que tem o intuito de ajudar usuários a entender como devem preencher determinado formulário, inserindo um texto contextual no campo a ser respondido

2- Na tag "input"com atributo type="radio" ou type="radio", para que serve o atributo "checked"?

R: Especifica que este botão "checkbox" ou "radio" é selecionado por default.

3- Na tag "input" para que serve o atributo "value"?

R: Especifica um valor padrão para um campo

4- Que atributo deve ser utilizado para agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez?

R: Checked

5- Para que serve a tag "label"? Para que serve o atributo "for" nessa tag?

R 1- Define a label para um elemento "input".

R 2- O atributo for da tag "label" tem uma forma de uso é que vc pode fazer vários tipos de customização pelo CSS escondendo o input default do browser por exemplo e usando a própria label como se fosse o botão.

6- Na tag "option" para que serve o atributo "selected"?

R: Este atributo booleano indica que várias opções podem ser selecionadas na lista. Se não for especificado, apenas uma opção poderá ser selecionada de cada vez

7- Nas tags de formulário para que serve o atributo "required"?

R: Especifica que um campo de entrada deve ser preenchido antes de enviar o formulário

8- Na tag "form" para que serve o atributo "action"?

R:Especifica para onde enviar os dados do formulário quando um formulário é enviado

9- Na tag "form" para que serve o atributo "method"?

R: Especifica o método HTTP a ser usado ao enviar dados de formulário

10- Para que o valor digitado de uma senha não apareça na barra de endereços do navegador, qual deve ser o valor do atributo "method" na tag "form"?

R: Hidden

11- Qual valor deve ser informado no atributo "method" na tag "form" em um formulário que será usado para fazer upload de arquivos?

R: Post
